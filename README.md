# DVWA Security Assessment Reports

This repository contains **DVWA vulnerability assessment reports**
written in an **industry-level / bug bounty style**.

## Vulnerabilities Covered
- API
- Authorisation Bypass
- Brute Force
- Command Injection
- Cryptography
- CSP Bypass
- CSRF
- File Inclusion
- File Upload
- Javascript
- SQL injection
- SQL injection (Blind)
- XSS (DOM)
- XSS (Reflected)
- XSS (Stored)


## Tools Used
- Burp Suite
- OWASP ZAP
- SQLMap
- Nmap
- Hydra
